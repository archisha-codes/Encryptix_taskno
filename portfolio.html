Projects

Smart Attendance System

The Smart Attendance System is a facial recognition-based attendance tracking solution developed to automate and streamline the attendance process. Utilizing Python's OpenCV library and machine learning algorithms, this system captures and recognizes student faces in real-time. The project aims to replace traditional attendance methods with a more efficient and accurate system. The application stores the attendance records in a database, providing easy access to attendance logs and generating reports.

Personal Finance Management System

The Personal Finance Management System is a web application designed to help users manage their finances effectively. Developed using HTML, CSS, JavaScript, and a backend powered by Python and Flask, this project allows users to track their income, expenses, and savings. The application features a dashboard that displays financial summaries and visualizations, helping users understand their spending habits and make informed financial decisions. Users can categorize transactions, set budget limits, and receive alerts when they are close to exceeding their budgets.
